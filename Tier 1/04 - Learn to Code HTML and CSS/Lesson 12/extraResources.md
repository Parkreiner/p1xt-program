# Extra Resources

## HTML & CSS

* [Mozilla Developer Network (super extensive documentation)](https://developer.mozilla.org/en-US/)
* [Dev.Opera (Seems to be more news- and features-based)](http://dev.opera.com/)
* [HTML Dog Tutorials (site design kind of leaves something to be desired, though)](http://www.htmldog.com/)
* [DevDocs (lets you quickly search through the documentation of any language)](http://devdocs.io)

## Design Inspiration

* [Dribbble](http://dribbble.com/)
* [Pattern Tap (now defunct)](http://patterntap.com/)
* [Premium Pixels (free, ready-made design elements)](http://www.premiumpixels.com)

## Frameworks

* [Bootstrap (needs no introduction)](http://getbootstrap.com/)
* [Foundation (used by sites like Amazon, Ebay, and The Washington Post)](https://foundation.zurb.com)
* [Skeleton Framework (literally never heard of this, but it seems super-lightweight)](http://getskeleton.com/)

## Style Guides

* [Web Style Guide (ugly, but seems to be packed with good information; seems to basically be a manual)](http://webstyleguide.com/wsg3/index.html)
* [Google's style guide for **writing** HTML and CSS](https://google.github.io/styleguide/htmlcssguide.html)
* [GitHub style guides (kinda seems abandoned, or at least repurposed)](https://github.com/styleguide/)

## Icons

* [Helveticons (extensive set of icons; not free)](http://hlvticons.ch/)
* [Ion icons (completely open-source)](http://ionicons.com/)
* [Fugue Icons (seem super '90s)](http://p.yusukekamiyamane.com)
* [Silk Icons (ditto)](http://www.famfamfam.com/lab/icons/silk/)
* [Pictos Icons (easily-modifiable free UI icons)](http://pictos.cc)
* [Picons (not free; seem kinda generic)](http://picons.me)
* [The Noun Project (very exensive and open-source/royalty-free)](https://thenounproject.com)

## Misc.

* [COLOUR Lovers (color palettes)](http://www.colourlovers.com/)
* [Color Hexa (color encyclopedia with programmatically-generated pairing recommendations)](https://www.colorhexa.com/)
* [Modernizr (provides information about what features browsers support)](https://modernizr.com)
* [jQuery (a library just like any other; devs don't need to rely on it as much nowadays, but it still has its uses)](http://jquery.com)
* [Google-Hosted Libraries (language libraries (mostly for JS) hosted by Google)](https://developers.google.com/speed/libraries/)
* [Copy-Paste Character (for copying-and-pasting obscure characters)](https://www.copypastecharacter.com)
